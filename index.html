# Tabs

Add a custom class to each item In this example I added a tab-content class

      <div class="tabs">
  <ul>
    <li class="is-active"><a>Pictures</a></li>
    <li><a>Music</a></li>
    <li><a>Videos</a></li>
    <li><a>Documents</a></li>
  </ul>
</div>
<div>
  <div class="tab-content is-block">Pictures Content</div>
  <div class="tab-content">Music Content</div>
  <div class="tab-content">Videos Content</div>
  <div class="tab-content">Documents Content</div>
</div>

Now add the following code to your styles:

      .tab-content {
  display: none;
}
